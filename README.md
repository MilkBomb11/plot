# PLOT
2d graph plotter. <br>
You can play around with it [here](https://milkbomb11.github.io/plot/).

## Examples
Exponential functions
```
y = e^x
```

Logarithmic functions
```
y = log(e, x)
```

Trigonometirc functions
```
y = tan x
```

Implicitly defined functions
```
x^2 + y^2 = 1
```

Multiple functions
```
x^2 + y^2 = 1 ; y = cos x ; y = sinx
```
